# Moya-Crnogora-Group---Corporate-Website

### A professional multi-page website developed for a real estate and residency consultancy firm.

## 📌 Project Overview
This project is a modern, responsive corporate website built for **Moya Crnogora Group**, a company specializing in real estate, residency, and consulting services in Montenegro (Karadağ) and Turkey. 

The website is designed to provide a smooth user experience, detailing the company's services, pricing, and corporate identity, while making it incredibly easy for potential clients to reach out via direct WhatsApp integration.

## 🚀 Key Features
- **Multi-Page Layout:** Structured architecture including Home, Services, About Us, and Pricing pages.
- **Smooth Animations:** Integrated **AOS (Animate On Scroll)** library for a dynamic, premium, and engaging feel as the user scrolls.
- **Direct Communication:** Built-in WhatsApp call-to-action buttons for instant client-to-business interaction.
- **Fully Responsive:** Optimized for a seamless experience across desktop, tablet, and mobile devices.
- **Modern UI Elements:** Clean aesthetic utilizing Boxicons for scalable and lightweight vector icons.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3
- **Libraries/Assets:** AOS (Animations), Boxicons
- **Concepts:** Responsive Web Design (RWD), UI/UX implementation.
